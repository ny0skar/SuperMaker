## Modelo de Negocio (Freemium)

### Gratis (Free)
- Máximo **1 tienda**
- Máximo **15 artículos** por visita
- Cada visita es **efímera** (no se guarda historial)
- **Sin** escaneo/conciliación de tickets
- **Sin** dashboard comparativo
- **Sin** modo offline (requiere internet siempre)

### Premium (Pago)
- **Tiendas ilimitadas**
- **Artículos ilimitados** por visita
- **Historial persistente** de todas las visitas
- **Escaneo de tickets** (OCR) + conciliación automática
- **Lectura de códigos de barras** desde datos del ticket
- **Dashboard** con comparativas y analytics de gastos
- **Tracking de fechas de caducidad**
- **Modo offline** con sincronización posterior

### Precio recomendado (App Store México)
- **Suscripción mensual:** $49 MXN/mes (~2.80 USD)
- **Suscripción anual:** $449 MXN/año (~25 USD — ahorro del 23%)
- **Nota:** Apple/Google se quedan con el 15-30%, así que el ingreso neto sería ~$34-42 MXN/mes y ~$314-381 MXN/año

> Estos precios son competitivos para el mercado mexicano. Apps similares como "Listonic" o "Bring!" cobran entre $39-$79 MXN/mes.

```

│   ├── src/
│   │   ├── routes/             # Express routes
│   │   ├── controllers/        # Lógica de negocio
│   │   ├── middleware/         # Auth JWT, rate limiting, plan validation
```

## Modelo de Datos (Prisma Schema Preview)

```prisma
model User {
  id            String    @id @default(uuid())
  email         String    @unique
  passwordHash  String
  displayName   String?
  avatarUrl     String?
  locale        String    @default("es")
  plan          Plan      @default(FREE)
  planExpiresAt DateTime?
  createdAt     DateTime  @default(now())
  updatedAt     DateTime  @updatedAt
  stores        Store[]
  visits        Visit[]
}

enum Plan {
  FREE
  PREMIUM
}

model Store {
  id        String   @id @default(uuid())
  name      String
  userId    String
  user      User     @relation(fields: [userId], references: [id])
  visits    Visit[]
  createdAt DateTime @default(now())
}

model Visit {
  id         String      @id @default(uuid())
  userId     String
  user       User        @relation(fields: [userId], references: [id])
  storeId    String
  store      Store       @relation(fields: [storeId], references: [id])
  status     VisitStatus @default(ACTIVE)
  total      Decimal     @default(0)
  ticketUrl  String?
  createdAt  DateTime    @default(now())
  finishedAt DateTime?
  items      VisitItem[]
}

enum VisitStatus {
  ACTIVE
  FINISHED
}

model VisitItem {
  id            String    @id @default(uuid())
  visitId       String
  visit         Visit     @relation(fields: [visitId], references: [id])
  name          String
  pricePerUnit  Decimal
  quantity      Decimal
  unit          Unit      @default(PIECE)
  subtotal      Decimal
  expiresAt     DateTime?
  barcode       String?
  createdAt     DateTime  @default(now())
}

enum Unit {
  PIECE
  KG
  G
  L
  ML
}
```
